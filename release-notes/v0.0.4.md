# Well Level Bridge v0.0.4

Removes configurable token index parsing completely.

The integration now parses only the numeric value immediately before the `Ft` unit, which matches the actual well sensor frame semantics. Existing config entries are migrated to remove legacy `primary_token_index` and `fallback_token_index` options. Each publish interval now averages every valid parsed sample received during that interval before updating the raw sensor and feeding the moving-average filter.
