# Well Level Bridge v0.0.2

Fixes frame parsing for the Node-RED replacement path.

The previous release used Python whitespace splitting, which collapsed repeated spaces in frames like `2026/06/19 11:38:45  0 D   74.27 Ft`. The original Node-RED flow used JavaScript `split(" ")`, preserving those empty fields. This release matches the Node-RED behavior so the default token indexes parse the well-level value correctly.
